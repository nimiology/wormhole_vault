SSH Key Setup

  1. Generate an SSH key on the backup PC:
    ssh-keygen -t ed25519 -f ./ssh_keys/id_ed25519 -N ""
  2. Copy the public key to the production server:
    ssh-copy-id -i ./ssh_keys/id_ed25519.pub user@your-server
  3. The private key is mounted into the container at /app/ssh_keys/id_ed25519
